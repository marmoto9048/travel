-----
橘通大街是心斋桥筋西边的一条商业街，这里不仅有人气网红品牌专卖店，更有容纳了全球各种小众潮牌的综合买手店。

祇园

清水寺
京都还有一处市场一直保留着，那就是大家熟知的【锦市场】。曾经是“京都的厨房”，京都代表市场。
如今被称作“京都厨房”的市场，可能还有几处。最著名的当属河原町金川出的「出町桝形商店街」、还残留着历史的痕迹。

金阁寺实在是比较坑的一个景点，建议避开。主要是游客太多了，而且阴天的时候金阁灰蒙蒙的，并不好看。去金阁寺的交通也不太方便，附近也没有太多其他可以顺路游览的景点（仁和寺、龙安寺都有一段路要走）。但是如果天气晴朗，蓝天白云，那金光闪闪的金阁寺十分值得去。如果幸运遇上雪后初霁，则绝对要去。

和服体验建议避开清水寺和祇园地区。一是游客实在太多，挤挤攘攘，二年坂三年坂全是坡道，走起来很累。同理，穿和服去伏见稻荷大社也是很考验脚力的。二是放眼望去随时能看见十几个和服变身的游客，虚荣心立刻大打折扣。建议可选：岚山大觉寺清凉寺地区、奈良、京都山科地区。

伏见稻荷大社不要只爬到千本鸟居就走回头路。继续往上爬，登顶并不吃力，但往上游客数量大减，且半山腰可以俯瞰京都市景，夕阳西下时景色最佳。

宇治的平等院，上午去比下午好。因为凤凰堂朝东，下午去的话，拍出来的照片全是大背光。平等院的博物馆里收藏的52尊云中供养菩萨像非常美，姿势各异、态度潇洒，真正极乐世界。

清水寺早上6点就开门了，在早上9点之前十分清净。祇园在晚上8点之后非常美，晚上10点左右还大概率能遇到下班的艺伎舞伎。不过请注意过分追逐舞伎、强行对着脸拍照，是很失礼的。

岚山渡月桥及附近早上9点前几乎没什么人，到了中午就太热闹了，旅行体验马上打折。

蹴上铁道樱花季即使六点去也都是人。只能拍近景，拍不了远景。建议找找其他更偏僻的樱花场所。

樱花和红叶季的行程原则：重点在于是否满开/见顷，而不是景点的名气。即使只有一棵樱花树满开的不知名寺庙，也比樱花已经开败全是秃枝的哲学之道要好。建议每天早上出发前根据当天花况调整行程。

夜樱和夜枫的话，关门前一小时去人比较少。如果晚上6点开门就去，光排队入场就要排很久。可以晚上8点去。高台寺的夜枫排队特别恐怖，可以去隔壁圆德院买套票，先参观圆德院，之后不用排队，直接进高台寺就好。

11月的红叶季除了红叶，还有银杏可赏。东本愿寺有大银杏树极美。

京都近郊很美。大原、鞍马、贵船、高雄、比叡、大津琵琶湖，都美，且游客通常比较少。

出町柳及再往北的鸭川，游客稀少，多是跑步、遛狗、放空的本地居民。

安倍晴明墓在岚电嵯峨站附近的居民区里。注意居民区严禁大声说话扰民。

本能寺在京都市役所附近，本能寺旧址在四条大宫附近，有一块石碑，旁边是本能寺消防局（这一定是故意的）。

龙马墓在京都灵山护国神社。円山公园内有龙马塑像。龙马遇刺的寺田屋在伏见。

10月22号白天是时代祭，晚上是鞍马地区的鞍马火祭。时间上几乎无缝衔接，要两个都去的话，非常考验体力。

个人推荐值得一试的和果子有：酱油团子（みたらし団子）、黄豆蕨饼（わらび餅）、红豆善哉、阿阇梨饼、栗饼（栗金団）。

中村藤吉的抹茶糖偏甜，伊藤久右卫门的抹茶糖偏苦。京都JR伊势丹楼上的中村藤吉，早上11点前去不用排队。

传统的和果子都不应该空口吃。配抹茶，一口和果子一口抹茶，和果子就不甜了，抹茶也不苦了，反而有甘甜的回味。

抹茶荞麦面坑爹，和普通荞麦面没区别。抹茶冰淇淋只有开头几口有抹茶味，味觉习惯后就是普通冰淇淋。

秋天的柿子、梨、栗子、菌菇好吃，春天的笋好吃。樱花口味的东西都不太好吃。



京都的寿司是鲭鱼寿司，不是生鱼片类的寿司，因为京都不靠海，传统上没有新鲜海鲜供应。

请尝一尝汤豆腐吧！无论哪家的都好吃。人间至美！

梅田（不含门票）

// ![夜景-梅田蓝天大厦](https://github.com/marmoto9048/travel/blob/master/%E6%A2%85%E7%94%B0%E8%93%9D%E5%A4%A9%E5%A4%A7%E5%8E%A6%E7%A9%BA%E4%B8%AD%E5%B1%95%E6%9C%9B%E5%8F%B0.jpg)

除了吃和买，还可以乘坐水上观光船，太左卫门桥下便是乘坐道顿堀水上观光船的码头（船费包含在“大阪周游卡”中）。活力四射的导游船员会让人立刻感受到关西人的开朗和热情。49194大阪城公园是以丰臣秀吉建造的大阪城天守阁为中心的城市公园，公园内可以看到各种街头艺人和聚会的大阪市民，而天守阁则可以俯瞰大阪全貌。

公园包括天守阁、西之丸庭院、丰国神社、市立修道馆、棒球馆以及一个露天音乐剧场。春季樱花盛开的时节，600多株樱花配合城堡灯光，美不胜收。

夜幕降临后更是欣赏大阪繁华夜景的绝佳场所。
由于梅田蓝天大厦的位置有点隐蔽，前往略微曲折。从大阪火车站5号出口出发，要穿过铁道下方的地下通道才可以到达。
